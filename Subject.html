Subject :

Subjects are basically related with the person who is doing the work. In English,there are
mainly two types of subjects. The first one is Singular subjects contains { He / She / It /
( Name of person / place / thing )} and the second one is plural subjects contains { I / We / You / They }.
                                                    

    Basically these are two types of subjects.

Singular SubjectsPlural Subjects
HeI
SheWe
ItYou
Name of person : RAMThey
Name of place : MANDSAUR
Name of thing : TABLE

Note :

* Singular verbs are used with singular subjects.
  Example : Ram(singular subject) is(singular verb) playing cricket.

* Plural verbs are used with plural subjects.
  Example : They(plural subject) are(plural verb) playing cricket.
