{ Future Tense }

Future tense is further divided into four types.

1. Future Simple

Use : It is basically used to show upcoming events.

Syntax : sub + will + verb(I) + obj.

Example : Ram will go to school.(upcoming event)

2. Future Continuous Tense

Use : Rarely used but whenever used, it is always used with an imagination of                future events.

Syntax : sub + will + be + verb(ing) + obj.

Here:(will) is the only helping verb which is used with both singular & plural subjects.

Example : Sita will be cooking food.(work in progress in future)

3. Future Perfect Tense

Use : It is basically used to show completion of any imagined future event.

Syntax : sub + will + have + verb(III) + obj.

Here:(will) is the only helping verb which is used with both singular & plural subjects.

Example : I will have completed this work.(work completion in future)

4. Future Perfect Continuous Tense

Use : It is basically used to show when an event is going on with a perticular time in          future.

Syntax : sub + will + have + been + verb(ing) + since/for + obj.

Here : (will) is the only helping verb which is used with both singular & plural subjects.

&

Since - is used with perfect time interval.(like 1990 / [02/12/2000] etc...)

For - is used with imperfect time interval.(like two months / two years etc...)

Example :
              Use of Since : I will have been playing cricket since I complete my work.(work in                                    progress with time interval in future)

              Use of For : We will have been dancing here for almost two hours.(work in                                 progress with time interval in future)
