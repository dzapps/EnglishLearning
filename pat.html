{ Past Tense }

Past tense is further divided into four types.

1. Past Simple

Use : It is basically used to show a perticular event of previous time.

Syntax : sub + verb(II) + obj.

Example : Ram went to school.(previous event)

2. Past Continuous Tense

Use : It is basically used where work is going on in a contiguous way in previous time.

Syntax : sub + was/were + verb(ing) + obj.

Here:
        (was) is used with singular subjects - he/she/it/name.

        (were) is used with plural subjects - I/we/you/they.

Example :
              Was - He was taking food.(work in progress in past)

              were - We were playing cricket.(work in progress in past)

3. Past Perfect Tense

Use : It is basically used when we want to show the completion of past event in                   previous time.

Syntax : sub + had + verb(III) + obj.

Here:(had) is the only helping verb which is used with both singular & plural subjects.

Example : We had played cricket.(work finished in past)

4. Past Perfect Continuous Tense

Use : It is basically used where work was going on in previous time in a contiguous way with a perticular time interval.

Syntax : sub + had + been + verb(ing) + since/for + obj.

Here : (had) is the only helping verb which is used with both singular & plural subjects.

Example :
              Use of Since : I had been living im mandsaur since 1990.(work in progress in past                                    for a perticular time)

              Use of For : He had been working on this project for almost two months.(work in                                 progress in past for a perticular time)
