{ Present Tense }

Present tense is further divided into four types.

1. Present Simple

Use : It is basically used for daily routine.

Syntax : sub + verb(I) + obj.

Example : Ram goes to school.(daily routine)

2. Present Continuous Tense

Use : It is basically used where work is going on in a contiguous way.

Syntax : sub + is/am/are + verb(ing) + obj.

Here:
        (is) is used with - he/she/it.

        (am) is used with - I.

        (are) is used with - we/you/they.

Example : Ram is going to school.(work in progress)

3. Present Perfect Tense

Use : It is basically used to represent the situation when daily works get finished.

Syntax : sub + has/have + verb(III) + obj.

Here:
        (has) is used with singular subjects - he/she/it/name.

        (have) is used with plural subjects - I/we/you/they.

Example :
              Has - He has taken food.(work finished)

              Have - I have finished my homework.(work finished)

4. Present Perfect Continuous Tense

Use : It is basically used where work is going on in a contiguous way with a perticular         time.

Syntax : sub + has/have + been + verb(ing) + since/for + obj.

Here :
        (has) is used with singular subjects - he/she/it/name.

        (have) is used with plural subjects - I/we/you/they.

&

Since - is used with perfect time interval.(like 1990 / [02/12/2000] etc...)

For - is used with imperfect time interval.(like two months / two years etc...)

Example :
              Use of Since : I have been living im mandsaur since 1990.(work in progress for                                    a perticular time)

              Use of For : He has been working on this project for almost two months.(work                                 in progress for a perticular time)
